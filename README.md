# Learning Deep Learning

## Overview
This repository contains concept-focused deep learning notebooks created as part of my learning journey by following the **CampusX** YouTube channel.  
The notebooks focus on understanding the internal working of neural networks, optimization techniques, and common challenges in deep learning through hands-on experiments.

## Concepts Covered
- Perceptron fundamentals and limitations
- Loss functions (hinge loss, classification vs regression losses)
- Backpropagation (classification and regression)
- Gradient Descent variants (Batch vs Stochastic)
- Feature scaling and its impact on training
- Weight initialization strategies
- Activation functions and initialization pitfalls
- Vanishing gradient problem
- Regularization techniques
- Dropout
- Batch normalization
- Early stopping
- Pooling operations in CNNs
- Visualization of CNN layers and activations

## Tools & Technologies
- **Python**
- **Jupyter Notebook**
- **NumPy**
- **Matplotlib / Seaborn**
- **TensorFlow / Keras**
- **scikit-learn** (supporting utilities)

## Repository Structure
- Each notebook focuses on a single deep learning concept
- Concepts are demonstrated through code, visualizations, and experiments
- Emphasis is placed on intuition and practical understanding rather than black-box usage

## Learning Outcomes
- Strong understanding of neural network fundamentals
- Practical insight into training instability and optimization challenges
- Ability to reason about model behavior and design choices
- Improved intuition for deep learning architectures and techniques
